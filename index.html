AI in games project (In progress)

For this project, I initially wanted to learn how to implement my own AI systems into games and ways to make realistic AI. This led me to take a semester-long (4 months) course where I could work on an AI system and attempt to implement all the systems I had been learning until this point.

Before starting, I decided to do as much research as possible on different ways that AI was being implemented in games and what were some important factors that I had to take into consideration. As I researched and started working on the finite state machine (FSM), I decided that I would only work on the FSM because it would allow me to make the best product possible.

Java Programming Advanced data structure

As part of a class I had taken at Brock University, I was able to work on and learn how to work with different data structures in Java. Throughout the assignments, I mostly worked on Double linked lists, Heaps, and similar data types. As a result, I learned distinctly about storing and manipulating data into arrays while understanding how memory and time complexity might affect a program when executed by the computer.
