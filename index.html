For this game/project I wanted to learn how AI for games are normally implemented and then attempt to implement my own system as a way to show what I have learnt over the project.

The first few months of the project I spent it doing reaserch learning about what what games use for AI and looking for tutorials that might be able to help me learn how to make a basic Finite State Machine (FSM) and a behaviour tree.
